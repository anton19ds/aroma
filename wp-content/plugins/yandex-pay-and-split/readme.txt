The official Yandex Pay and Split module for WooCommerce

== Description ==

**Важно**

Для корректной работы модуля в WooCommerce обязательно должно быть настроено значение постоянной ссылки отличное от значения "Обычная"

**Описание**

Плагин Яндекс Пэй и Сплит — платежное решение для интернет-магазинов на базе WooCommerce, которое позволит вашим клиентам оплачивать покупки сразу с кешбэком баллами Плюса или частями с помощью Яндекс Сплит.

Преимущества:
* Быстрая интеграция: Простой процесс установки и настройки позволяет начать прием платежей в кратчайшие сроки.
* Безопасность: Надёжно храним данные банковских карт клиентов.
* Удобство: При оплате платежные данные заполняются автоматически за один клик.
* Широкая база пользователей: Миллионы пользователей Яндекса станут вашими потенциальными клиентами.

Основные возможности:
* Получение ссылки на оплату
* Полная оплата с кешбэком баллами Плюса
* Оплата частями
* Полный и частичный возврат средств в личном кабинете Яндекс Пэй

Поддерживаемые способы оплаты:
* Банковские карты
* СБП
* Карта Пэй

Подробности в [нашей документации](https://pay.yandex.ru/docs).

**Important**

For the module to work correctly in WooCommerce, a permalink value other than the "Plain" value must be configured.

**Description**

The Yandex Pay and Split plugin is a payment solution for WooCommerce—based online stores that will allow your customers to pay for purchases immediately with cashback Plus points or in installments using Yandex Split.

Advantages:
* Fast integration: A simple installation and configuration process allows you to start accepting payments as soon as possible.
* Security: We securely store customer's bank card data.
* Convenience: When paying, payment details are filled in automatically in one click.
* Wide user base: Millions of Yandex users will become your potential customers.

Main features:
* Receiving a payment link
* Full payment with cashback Plus points
* Payment in installments
* Full and partial refund in your Yandex Pay account

Supported payment methods:
* Bank cards
* SBP
* Yandex Pay Card

The details are in [our documentation](https://pay.yandex.ru/docs).

== Installation ==
= Automatic installation =

Для автоматической установки следуйте [инструкции](https://pay.yandex.ru/docs/ru/cms/wordpress/#setup) из нашей документации.

= Manual installation =

1. Скачайте архив с маркетплейса WordPress или на официальном сайте Яндекс.
2. Распакуйте скачанный архив.
3. Скопируйте папку `yandex_pay_and_split` в директорию `wp-content/plugins/` на вашем сервере.

== Сonfiguration ==

Для настройки плагина следуйте [инструкции](https://pay.yandex.ru/docs/ru/cms/wordpress/#settings) из нашей документации.
